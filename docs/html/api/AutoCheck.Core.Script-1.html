Class Script<T>

This class must be inherited in order to develop a generic custom script. The script is the main container for a set of instructions, which will test the correctness of an assignement.

Batch()

This method will loop through a set of students assignments, running the script for all of them. All the assignments will be unziped, loaded into the local copy detector, and all the data from previos executions will be restored prior the script execution. If a potential copy is detected, the script execution will be skipped.
