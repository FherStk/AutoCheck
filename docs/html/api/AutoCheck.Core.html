Namespace AutoCheck.Core

Classes

Checker

This class must be inherited in order to develop a custom checker. The checker is in charge of testing items using a connector, and the result will be always a list of errors.

Connector

This class must be inherited in order to develop a custom connectors. This class is an abstraction layer between a checker (to a lesser extent, a script) in order to perform in/out operations and/or data validations.

CopyDetector

This class must be inherited in order to develop a custom copy detectors. This class is in charge of performing the copy detection along the student's files, code the abstract methods and provide all the necessary extra code needed.

Output

This class is in charge of writing the output into the terminal.

Script<T>

This class must be inherited in order to develop a generic custom script. The script is the main container for a set of instructions, which will test the correctness of an assignement.

ScriptDB<T>

This class must be inherited in order to develop a database oriented custom script. The script is the main container for a set of instructions, which will test the correctness of an assignement.

ScriptFiles<T>

This class must be inherited in order to develop a file-system oriented custom script. The script is the main container for a set of instructions, which will test the correctness of an assignement.
